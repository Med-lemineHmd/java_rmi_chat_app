/* Compile all Java files at once */
```
javac *.java
```
/* Then Start the rmi registry using the following command */
```
start rmiregistry
```
/* Run the server */
```
java -Djava.security.policy=security.policy ChatServer
```
/* Run the client */
```
java -Djava.security.policy=security.policy ChatClient
```
